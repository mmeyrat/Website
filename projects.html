OuistitiBot

Un bot de discussion codé avec Discord.js 14. Il répond à des messages précis pour mettre de l'ambiance dans vos serveurs Discord.

MiniGolf VR

Un jeu en réalité virtuelle de mini-golf réalisé avec Unity. Il consiste en un parcours de 3 trous. Ce jeu est destiné avant tout à l'Occulus Quest.

CoeurBot

Un bot de discussion codé avec Python 3 et basé sur TwitchIO. Il comprend diverses commandes pour animer un stream Twitch.

Twitch Emotes Extension

Une extension de navigateur web réalisée avec Manifest v3 pour Chrome et Manifest v2 pour Firefox. Elle ajoute des emotes personnalisées dans un chat Twitch.

Earth 3D

Une page web contenant une vue 3D de la planète Terre conçue avec la bibliothèque Three.js.
